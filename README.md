## 🚀 快速開始

### 1. 環境準備

```bash
# 需要的工具
- n8n (v1.115.3 或以上)
- Node.js 18+
- Git
```

### 2. 匯入 Workflow

1. 選擇你需要的工作流分類
2. 下載對應的 `.json` 文件
3. 在 n8n 介面點選「Import from File」
4. 根據各專案的 README 設定憑證
5. 啟用並測試 workflow

### 3. 設定指南

每個分類資料夾都包含：
- 📄 `README.md` - 詳細說明文件
- 📦 `*.json` - n8n workflow 檔案
- 🔧 設定步驟與範例
